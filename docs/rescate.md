# Rescate del material original

El ZIP original contenía ejercicios sueltos de fundamentos de programación, prácticas de consola, retos tipo URI/beecrowd, archivos compilados `.class`, configuraciones de VS Code y borradores.

Para este repositorio se conservaron las ideas principales y se reorganizaron como código limpio:

- Variables y tipos de datos.
- Operadores.
- Casting.
- Entrada por consola.
- Cálculos geométricos.
- Conversiones.
- Condicionales.
- Ciclos.
- Retos de clasificación numérica.
- Mini aventura de consola.

## Limpieza aplicada

No se conservaron:

- Archivos `.class`.
- Configuraciones locales de `.vscode`.
- Workspaces locales.
- Borradores `.txt`.
- Archivos vacíos.
- Nombres de clases ambiguos.
- Duplicados con `Main`.
- Archivos con nombres inválidos para Java.

## Resultado

El material quedó como un laboratorio de consola ejecutable desde un menú principal, con paquetes organizados y documentación.
