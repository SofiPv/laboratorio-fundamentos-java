# Catálogo de ejercicios

## Fundamentos

| Clase | Tema | Descripción |
|---|---|---|
| `VariablesDemo` | Variables | Declara variables y muestra sus valores. |
| `TiposDatosDemo` | Tipos primitivos | Presenta `byte`, `short`, `int`, `long`, `float`, `double`, `char`, `boolean` y `String`. |
| `OperadoresDemo` | Operadores | Realiza suma, resta, multiplicación, división y módulo. |
| `CastingDemo` | Conversiones | Muestra casting explícito, implícito y redondeo. |
